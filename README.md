## Opis atributa podataka

| Naziv atributa | Opis |
|----------------|------|
| **naziv** | Puni naziv dobrovoljnog vatrogasnog društva |
| **adresa** | Objekt koji sadrži lokacijske podatke DVD-a |
| • ulica | Naziv ulice i kućni broj sjedišta društva |
| • poštanskiBroj | Poštanski broj područja |
| • grad | Naziv grada ili naselja |
| **kontaktEmail** | Službena e-mail adresa društva za kontakt |
| **webStranica** | Službena web stranica društva |
| **OIB** | Jedinstveni identifikacijski broj društva |
| **predsjednik** | Osoba koja vodi administrativno poslovanje društva |
| • ime / prezime / kontaktBroj | Podaci o predsjedniku |
| **zapovjednik** | Osoba odgovorna za operativne i vatrogasne aktivnosti |
| • ime / prezime / kontaktBroj | Podaci o zapovjedniku |
| **tajnik** | Osoba zadužena za administrativne i evidencijske poslove |
| • ime / prezime / kontaktBroj | Podaci o tajniku |
| **godinaOsnutka** | Godina kada je društvo osnovano |
| **brojČlanova** | Ukupan broj aktivnih i pridruženih članova DVD-a |
